Program Name: Connect Four
Difficulty: 7

Explanation: Our program renders a two player Connect Four game with the default dimensions of 7 columns by 6 rows. In order to win, one must have their drops, either red or yellow, aligned 4 in a row. Furthermore, our game allows you to customize the dimensions of the board based on an input your choice.
